# Reading voice-model licenses

Voice projects often ship several things under different terms. This catalog separates the code license from the model-weight license for that reason.

Check all of these before shipping a product:

- Inference and training code
- Model weights and fine-tuned derivatives
- Training or evaluation datasets
- Included voice packs
- Reference recordings used for cloning
- Generated-output terms, if the publisher states any
- Names, trademarks, and likeness rights

`Commercial: permissive` means the cited code and weight terms do not state a non-commercial restriction. It is not legal advice and does not grant rights to a speaker's identity or to third-party input material.

`Conditional` means use may be possible but requires obligations or voice-specific review. `Restricted` means the cited terms state a non-commercial, research-only, or similar limitation. `Varies` means checkpoints or voices differ.

When code and weights conflict, the stricter applicable terms usually control the combined distribution. Ask qualified counsel when a license affects a commercial release.
