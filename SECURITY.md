# Security and privacy

Do not open a public issue for leaked credentials, unsafe model-loading behavior, private recordings, or personal voice data.

Use GitHub's private vulnerability reporting for this repository. Include the affected file, a minimal reproduction, and the impact. Do not attach a real person's voice recording. A synthetic test artifact is preferred.

This repository is a catalog and static site. Security issues in a listed model or app should also be reported to that project's maintainers.
